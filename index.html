How it works

1. You send the document

You send your document (PDF, image, scan, screenshot) via WhatsApp, Telegram or email. No portals, no accounts, no friction.

2. We normalize the content

We convert the file into a canonical, stable representation (content normalization), so that compression or metadata changes do not affect verification.

3. We generate the hash

A SHA‑256 hash is generated from the normalized content. This hash uniquely represents the certified version of your document.

4. We notarize on Ethereum

The hash is recorded on Ethereum mainnet, creating an immutable, time‑stamped proof of existence and integrity.

5. You receive a verification report

You receive a PDF or JSON report including:

  • Document hash
  • Timestamp and block number
  • Ethereum transaction ID
  • Verification instructions

Optional: we can also mint an NFT that represents the certified document.
